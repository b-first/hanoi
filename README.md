# Hanoi Game

- Download main.py and CPU_Player.py
- Set the level variable to what positive number you'd like
- Run and play
    - Left arrow selects the left stack
    - Down arrow selects the middle stack
    - Right arrrow selects the right stack
    - Enter (Return) key resets the game
    - c key resets and solves as the CPU player
    - h stops the CPU (if playing) and lets the player play
